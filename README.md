# Friend4Friend

# Workflow

## 1 Prerequirements
  Sails 0.10-rc8 ```$ npm install -g sails@beta```
  Bower ```$ npm install -g bower```
  Grunt ```$ npm install -g grunt```

## 2 Install dependencies
  ```$ npm install && bower install ```

## 3 Build assets
  ```$ grunt build ``` 

## 4 Run the app
  ```$ sails lift ```
